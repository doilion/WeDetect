## 关键结论

### 最优配置

| 目标 | 推荐实验 | mAP |
|------|---------|-----|
| Base 类别最高 | **exp7** (英文 + 强增强) | 31.4% |
| Novel 泛化最好 | **exp3** (Loss 调整) | 9.1% |
| 综合最优 | **exp7** | Base 31.4%, Novel 8.6% |

### 实验发现

1. **强数据增强有效**: exp7 (强增强) 比 exp5 (弱增强) Base mAP 提升 3.0%
2. **英文文本略优**: exp7 (英文) 比 exp6 (中文) Base mAP 提升 1.2%
3. **Loss 调整提升泛化**: exp3 的 Novel mAP (9.1%) 是所有实验中最高
4. **冻结 Backbone 不推荐**: exp4 组合策略 Base 下降严重 (-7.8%)

## 常用命令

### 训练
```bash
# 单卡训练
CUDA_VISIBLE_DEVICES=0 python train.py config/wedetect_tiny_tct_exp7.py

# 多卡分布式
bash dist_train.sh config/wedetect_tiny_tct_exp7.py 4
```

### Base 评估 (排除 Negative)
```bash
PYTHONPATH=. python test_exclude_negative.py \
  --config config/wedetect_tiny_tct_exp7.py \
  --checkpoint work_dirs/wedetect_tiny_tct_exp7/best_coco_bbox_mAP_epoch_10.pth \
  --work-dir work_dirs/eval/test_base_exclude_neg_exp7
```

### Novel 评估
```bash
PYTHONPATH=. python eval_novel.py \
  --config config/wedetect_tiny_tct_exp7.py \
  --checkpoint work_dirs/wedetect_tiny_tct_exp7/best_coco_bbox_mAP_epoch_10.pth \
  --text data/texts/tct_ngc_v2_class_texts_en.json \
  --out-dir work_dirs/eval/test_novel_exp7
```
